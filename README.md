# 大龙山工具间

> **代码生成声明**：本项目 80% 代码由 Trae 生成，20% 由 Claude Code + DeepSeek 生成，5% 由古法手工编程但已被生成式模型重写覆盖。

面向空间数据处理、文档生成、数据查询的在线工具集。纯前端，即开即用。

所有工具均为纯 JS 实现——数据处理和 API 请求全部在浏览器端完成，无需后端服务器。这个范式的核心优势：**零服务端成本，静态文件部署即可分发，一个 `npm run build` + 任意静态托管 = 全球可用**。

## 技术栈

Vue 3 + TypeScript + Tailwind CSS + daisyUI / Vite / Pinia

## 开发

```sh
npm install
npm run dev      # 启动开发服务器
npm run build    # 生产构建
```

## 工具列表

坐标转换 · 批量坐标转换 · 距离测算 · 地址解析 · 经纬度查询 · 点转 KML · 基站转 KML · 空间分析 · GIS 转表格 · SHP 转表格 · DOCX 模板导出 · XLSX 模板导出 · 模板导出 · KML 查看器 · SHP 转 KML · 甘特图 · 高程查询 · 批量高程查询 · 历史天气查询 · 地震查询 · GDACS 灾害数据 · 二维码工具
