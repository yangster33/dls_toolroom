# 器

## 工具简介

高程查询工具用于通过经纬度坐标查询对应位置的海拔高度，数据来源于 Open-Meteo Elevation API，支持全球范围内的高程查询。

## 功能特点

- **全球覆盖**：支持查询全球任意位置的高程数据
- **精度可靠**：基于 SRTM 90m 精度数字高程模型
- **快速响应**：API响应迅速，查询体验流畅
- **跨域支持**：支持前端直接调用，无需后端代理

## 使用方法

1. 在「纬度」输入框中输入目标位置的纬度值（范围：-90 至 90）
2. 在「经度」输入框中输入目标位置的经度值（范围：-180 至 180）
3. 点击「查询高程」按钮
4. 系统将显示查询结果，包括原始坐标和对应的海拔高度

## 输入格式

- **纬度**：数值类型，范围 -90 至 90
- **经度**：数值类型，范围 -180 至 180

## 输出格式

查询结果以 JSON 格式展示：
```json
{
  "latitude": 39.9042,
  "longitude": 116.4074,
  "elevation": 44.4
}
```

同时以卡片形式展示各字段的具体值。

## 注意事项

1. 请确保输入的经纬度值在有效范围内
2. 高程数据基于 SRTM 90m 精度，适用于一般应用场景
3. 部分区域可能存在数据缺失或精度较低的情况
4. 高程值以米为单位，负值表示低于海平面
5. API有请求频率限制，请合理使用
