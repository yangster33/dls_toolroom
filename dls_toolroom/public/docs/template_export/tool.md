# 器

## 工具简介

模板批量导出工具能够根据您提供的表格数据和文档模板，自动生成多份格式统一的文档文件。适用于需要批量制作合同、证书、通知、报告、标签、信函等重复性文档的场景。

您只需准备一份包含所有数据的表格（CSV 或 Excel），以及一份设计好版式的文档模板（Word 或 Excel），工具会自动为表格中的每一行数据生成对应的文档。

## 支持的输入格式

| 类型 | 格式 | 说明 |
|------|------|------|
| 数据文件 | CSV（.csv） | 逗号分隔的纯文本表格，第一行为列名 |
| 数据文件 | Excel（.xlsx / .xls） | 电子表格，第一个工作表的第一行为列名 |
| DOCX 模板 | Word（.docx） | 使用 `{字段名}` 作为占位符 |
| XLSX 模板 | Excel（.xlsx） | 使用 `${字段名}` 作为占位符 |

## 操作步骤

### 1. 下载数据模板

点击"下载模板"按钮，选择 CSV 或 Excel 格式。下载的模板包含示例字段名（name、age、city），您可以将其替换为自己的实际字段名和数据内容。

### 2. 上传数据文件

上传填写好的数据文件。工具会自动读取并显示文件中有多少行数据，以及检测到哪些字段名。这些字段名将用于在模板中匹配占位符。

### 3. 设置忽略行数

指定从数据文件的第几行开始读取。默认为 1（即跳过第 1 行，从第 2 行开始），适用于第一行是表头的情况。如果您的数据文件前若干行是说明文字，可以增大此数值。

### 4. 上传文档模板

上传设计好的文档模板文件（DOCX 或 XLSX）。模板中需要使用占位符标记数据填充位置：
- Word 模板中，使用 `{字段名}` 的格式
- Excel 模板中，使用 `${字段名}` 的格式

上传模板后，工具会自动列出模板中检测到的字段，点击字段名按钮可将该字段添加到输出文件名模板中。

### 5. 设置输出文件名

- **文件名模板**：定义输出文件的命名规则，支持 `{字段名}` 占位符。例如 `{name}-{日期}` 会生成类似"张三-2024-01-15"的文件名。留空则自动使用数据第一列的值作为文件名。
- **重名处理策略**：
  - **追加序号**：当多行数据产生相同文件名时，在重名文件后追加 (1)、(2) 等序号
  - **行号前缀**：为所有文件名统一添加等宽行号前缀（如 001-、002-），确保完全无重名

设置后会实时预览前 3 个文件名的效果，并提示是否存在重名冲突。

### 6. 渲染并下载

点击"渲染"按钮，工具开始逐行生成文档。进度条会显示当前处理进度。渲染完成后：
- 如果只有 1 份文档，直接下载该文件
- 如果有多份文档，打包为一个 ZIP 压缩文件供一次性下载

## 模板占位符高级用法

在 Word 模板（DOCX）中，占位符支持过滤器语法，使用管道符 `|` 分隔：

| 过滤器 | 用法示例 | 效果 |
|--------|---------|------|
| `upper` | `{name \| upper}` | 转为大写 |
| `lower` | `{name \| lower}` | 转为小写 |
| `capitalize` | `{name \| capitalize}` | 首字母大写 |
| `number` | `{price \| number:2}` | 保留 2 位小数 |
| `currency` | `{price \| currency}` | 货币格式（默认￥） |
| `percent` | `{rate \| percent:1}` | 百分比格式（乘 100 后保留 1 位） |
| `date` | `{date \| date:'yyyy年mm月dd日'}` | 日期格式化 |
| `default` | `{name \| default:'未填写'}` | 空值时使用默认值 |
| `truncate` | `{desc \| truncate:20}` | 超过 20 字符截断并加省略号 |
| `trim` | `{name \| trim}` | 去除首尾空格 |
| `split` | `{name \| split:' ':0}` | 按分隔符拆分后取第 N 段 |

### 内置特殊变量

模板中可以使用以下特殊变量，无需在数据文件中定义：

| 变量 | 说明 |
|------|------|
| `{rowIndex}` | 当前行在数据中的序号（从 1 开始） |
| `{totalRows}` | 数据总行数 |
| `{items}` | 全部数据的 JSON 字符串表示 |

## 注意事项

1. **字段名大小写敏感**：模板中的占位符字段名必须与数据文件的列名完全一致（包括大小写和中英文）。建议统一使用英文字段名以避免编码问题。

2. **占位符语法因模板类型而异**：Word 模板使用 `{字段名}`，Excel 模板使用 `${字段名}`。混用会导致占位符无法识别。

3. **Excel 日期数据**：在 Excel 数据文件中，日期列的格式会被自动识别和转换。如果您的日期数据显示异常，请检查数据源中日期列的单元格格式是否为"日期"而非"文本"。

4. **忽略行数不能导致数据为空**：假设数据文件共 10 行（含表头），设置忽略前 10 行后将没有剩余数据供渲染，操作会报错。请确保忽略行数设置后仍有至少 1 行有效数据。

5. **模板文件不可为空**：上传的模板文件必须包含至少一个占位符或内容。如果模板文件损坏或格式不正确，渲染阶段会报错，请检查模板文件是否能正常用 Office 软件打开。

6. **Excel 模板中的单元格格式会保留**：在 XLSX 模板渲染时，如果占位符占据整个单元格且替换值为数字或日期类型，原始单元格的数字格式（如货币、百分比、日期格式）会被保留。但字符串内嵌替换（如"姓名：{name}"）不会保留格式。

7. **批量渲染耗时与数据量成正比**：每行数据都需要独立完成一次完整的模板加载和渲染过程。数据行数较多时（如超过 500 行），请耐心等待，不要中途关闭页面或刷新浏览器。

8. **文件名非法字符会被自动替换**：如果数据中包含文件名不允许的字符（`\/:*?"<>|`），工具会自动将其替换为短横线 `-`，确保文件能正常保存到您的电脑。
